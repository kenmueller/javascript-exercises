# JavaScript Exercises

**Implement JavaSript functions from the ground up**

## Initialization

1. Clone the repo
2. Run `npm install` (if you get an error, run `npm i -g npm`)
3. Run `cd exercises`

```bash
npm test {filename}
```

## Exercises

**Easy**

1. map
2. filter
3. uniq

**Medium**

1. flatten
2. reduce
3. break

**Hard**

1. sample
2. safeGet
3. memoize

**Extra Hard**

1. curry
2. debounce

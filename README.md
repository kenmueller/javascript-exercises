# JavaScript Exercises

**Implement JavaSript functions from the ground up**

## How to use it

1. Clone the repo.
2. Run `npm install`
3. Run `npm run test` and watch the tests fail.
4. Fix the tests one by one.

## Exercises

**Easy**

1. map
2. filter
3. uniq

**Medium**

1. flatten
2. reduce
3. break

**Hard**

1. sample
2. safeGet
3. memoize

**Extra Hard**

1. curry
2. debounce
